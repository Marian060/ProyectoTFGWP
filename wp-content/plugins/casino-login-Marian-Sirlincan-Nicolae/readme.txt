Plugin educativo que añade un sistema de login, registro y perfil personalizado para Casino M.N.S, integrado con el sistema de fichas.

== Shortcodes ==

[mns_login_form]     - Muestra el formulario de inicio de sesión.
[mns_register_form]  - Muestra el formulario de registro (con fecha de nacimiento y términos).
[mns_profile_page]   - Muestra la página de perfil con avatar, datos básicos y el historial de fichas.
[mns_auth_menu]      - Muestra el botón de login/registro o el menú desplegable del usuario con logout e historial.
[mns_logout_button]  - Muestra un botón de cerrar sesión, con atributos opcionales redirect_to y label.
